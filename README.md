# Sales Analysis Using Python

## Project Overview

This project performs an Exploratory Data Analysis (EDA) on a sales dataset using Python to uncover customer purchasing behavior and sales trends. The analysis focuses on cleaning the dataset, exploring customer demographics, and visualizing sales performance across different dimensions such as age group, gender, and geographical zones.

---

## Objective

The objectives of this project are to:

- Clean and preprocess the sales dataset.
- Analyze customer demographics.
- Understand purchasing patterns based on age and gender.
- Compare sales performance across different zones.
- Generate business insights through data visualization.

---

## Tools and Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset Information

The dataset contains customer sales information, including:

- Customer Name
- Gender
- Age
- Age Group
- Orders
- Sales Amount
- Zone
- Product Details

---

## Data Preprocessing

The following preprocessing steps were performed:

- Imported the dataset using Pandas.
- Examined the dataset structure and data types.
- Removed unnecessary columns.
- Handled missing values by removing null records.
- Renamed column names for better readability.
- Generated descriptive statistics for numerical columns.

---

## Exploratory Data Analysis

### 1. Age Group Distribution

A count plot and pie chart were created to analyze customer distribution across different age groups.

**Observation**

- The analysis identifies the age groups contributing the highest number of customers.

---

### 2. Gender Distribution

Customer distribution was analyzed based on gender using count plots and pie charts.

**Observation**

- The dataset shows the proportion of male and female customers, helping understand the customer base.

---

### 3. Sales by Gender

A bar chart was used to compare the total sales amount generated by each gender.

**Observation**

- The visualization highlights which customer segment contributes more to total revenue.

---

### 4. Sales by Zone

Sales performance was analyzed across different geographical zones using bar charts.

**Observation**

- The dashboard identifies the zones generating the highest sales revenue, enabling regional performance analysis.
